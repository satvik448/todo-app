# To-Do List App

This is a simple to-do list web application built with HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, and delete them. Tasks are saved in the browser's local storage so they persist between sessions.

## Features

- Add new tasks
- Mark tasks as completed (strikethrough)
- Delete tasks
- Data persistence using localStorage

## How to Run

1. Clone or download the project files.
2. Open `index.html` in a web browser.
3. Alternatively, run a local server (e.g., using Python: `python -m http.server`) and navigate to `http://localhost:8000`.

## File Structure

- `index.html`: The main HTML file
- `css/style.css`: Styles for the app
- `js/script.js`: JavaScript functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

This project was created as a learning exercise for web development basics.
